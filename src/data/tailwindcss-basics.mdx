# Tailwind CSS Basics: A Complete Guide

Tailwind CSS is a utility-first CSS framework that helps you build modern, responsive websites faster. Instead of writing custom CSS, you apply pre-defined utility classes directly to your HTML elements.

## What is Tailwind CSS?

Tailwind CSS is a low-level utility framework that gives you building blocks to create any design without leaving your HTML. It's different from traditional CSS frameworks like Bootstrap because it follows a **utility-first** approach.

### Key Concepts

**Utility-First Approach**: Rather than creating custom CSS classes, you compose your styles using single-purpose utility classes.

```html
<!-- Traditional CSS -->
<div class="card"><!-- needs custom CSS --></div>

<!-- Tailwind CSS -->
<div class="m-4 rounded-lg bg-white p-6 shadow-md">
  <!-- styles applied directly via utilities -->
</div>
```

## Getting Started

### Installation

Install Tailwind CSS in your project:

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

Then configure your `tailwind.config.js`:

```javascript
export default {
  content: ["./src/**/*.{js,ts,jsx,tsx}"],
  theme: {
    extend: {},
  },
  plugins: [],
};
```

### Basic Setup

Add Tailwind directives to your global CSS:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

## Core Utility Classes

### Display & Layout

- `flex` - Creates a flex container
- `grid` - Creates a grid container
- `block`, `inline-block`, `inline` - Display properties
- `hidden` - Hides elements

### Spacing

- `p-*` - Padding (p-4, p-8, etc.)
- `m-*` - Margin (m-2, m-6, etc.)
- `gap-*` - Gap between flex/grid items

### Colors

```html
<!-- Text color -->
<p class="text-red-600">Red text</p>

<!-- Background color -->
<div class="bg-blue-500">Blue background</div>

<!-- Border color -->
<button class="border-2 border-green-400">Green border</button>
```

### Typography

- `text-sm`, `text-base`, `text-lg`, `text-xl` - Font sizes
- `font-light`, `font-normal`, `font-bold` - Font weights
- `text-center`, `text-left`, `text-right` - Text alignment

### Responsive Design

Tailwind uses breakpoints to create responsive designs:

- `sm` - 640px
- `md` - 768px
- `lg` - 1024px
- `xl` - 1280px

```html
<!-- Single column on mobile, 2 columns on medium screens -->
<div class="grid grid-cols-1 gap-4 md:grid-cols-2">
  <div>Item 1</div>
  <div>Item 2</div>
</div>
```

### Hover & States

```html
<!-- Hover effects -->
<button class="bg-blue-500 transition hover:bg-blue-700">Hover me</button>

<!-- Focus states -->
<input class="border-2 border-gray-300 focus:border-blue-500" />
```

## Common Patterns

### Button

```html
<button
  class="rounded-lg bg-blue-600 px-4 py-2 font-semibold text-white transition duration-200 hover:bg-blue-700"
>
  Click me
</button>
```

### Card

```html
<div class="max-w-sm rounded-lg bg-white p-6 shadow-md">
  <h3 class="mb-2 text-xl font-bold">Card Title</h3>
  <p class="text-gray-600">Card description goes here.</p>
</div>
```

### Navigation Bar

```html
<nav class="flex items-center justify-between bg-gray-900 p-4 text-white">
  <div class="text-lg font-bold">Logo</div>
  <div class="flex gap-4">
    <a href="#" class="hover:text-gray-300">Home</a>
    <a href="#" class="hover:text-gray-300">About</a>
    <a href="#" class="hover:text-gray-300">Contact</a>
  </div>
</nav>
```

## Tips & Best Practices

1. **Use Responsive Prefixes** - Design mobile-first and add breakpoints progressively
2. **Extract Components** - For repeated patterns, use `@apply` to create component classes
3. **Custom Configuration** - Extend the default theme for brand colors and spacing
4. **Purging** - Tailwind automatically removes unused CSS in production
5. **Dark Mode** - Enable dark mode in your config and use `dark:` prefix

### Using @apply for Components

```css
@layer components {
  .btn-primary {
    @apply rounded-lg bg-blue-600 px-4 py-2 font-semibold text-white transition hover:bg-blue-700;
  }
}
```

## Advantages of Tailwind CSS

✅ Faster development - No need to switch between HTML and CSS files  
✅ Smaller bundle size - Unused styles are automatically purged  
✅ Consistency - Pre-defined spacing, colors, and sizing scales  
✅ Responsive by default - Mobile-first design is built-in  
✅ Easy to maintain - Styles are co-located with markup

## Conclusion

Tailwind CSS is a powerful tool that revolutionizes how you write CSS. By using utility classes, you can build beautiful, responsive designs faster than ever. Start with the basics, explore the extensive documentation, and you'll be building amazing UIs in no time!
